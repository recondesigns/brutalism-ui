## Getting Started

### Install package

```sh
npm install brutalism-ui
```

### Usage

In the consuming app, import the components as needed.

```jsx
import { Button } from "brutalism-ui"

export const Form = () => {
  return (
    <form>
      ...otherElements
      <Button label="Submit" />
    </form>
  )
}
```
